# 塾スケジューラー かんたんインストールガイド(Windows)

このガイドは、**パソコンにくわしくない方向け**に書かれています。
「Python って何?」「コマンドって何?」という状態でも、上から順番に
やっていけば必ず動かせます。所要時間はだいたい 15〜20 分です。

## これは何のソフト?

塾の時間割を自動で作ってくれるソフトです。生徒・先生・教科・教室・
出席できる日時を登録すると、条件(先生は同時に2人まで、生徒は1日
1コマまで、など)を守った時間割を自動で組み、ブラウザ(Edge や
Chrome)の画面で確認・修正・印刷ができます。

データはすべて**自分のパソコンの中だけ**に保存されます。インター
ネット上に送信されることはありません。

## 全体の流れ

1. Python(パイソン)というソフトを入れる … 最初の1回だけ
2. このソフトをダウンロードして展開する … 最初の1回だけ
3. `setup.bat` をダブルクリック … 最初の1回だけ
4. `start.bat` をダブルクリック … **使うときは毎回これだけ**

---

## 手順1: Python を入れる(最初の1回だけ)

Python は、このソフトを動かすための「エンジン」にあたるものです。
無料で、公式サイトから安全に入手できます。

1. ブラウザで <https://www.python.org/downloads/> を開きます。
2. 黄色い「**Download Python 3.x.x**」ボタンを押します。
   (3.11 以上ならどのバージョンでも大丈夫です)
3. ダウンロードされたファイル(`python-3.x.x-amd64.exe`)を
   ダブルクリックします。
4. **ここが一番大事です:** 最初の画面の下にある
   「**Add python.exe to PATH**」というチェックボックスに
   **必ずチェックを入れてから**、「Install Now」を押します。
   - チェックを忘れると後の手順が失敗します。忘れてしまった場合は
     いったんアンインストールして入れ直すのが簡単です。
5. 「Setup was successful」と出たら「Close」で閉じます。

## 手順2: このソフトをダウンロードする(最初の1回だけ)

1. ブラウザで
   <https://github.com/takanesano/school-scheduler> を開きます。
2. 緑色の「**Code**」ボタン →「**Download ZIP**」を押します。
   (こちらの直接リンクでも同じものが落とせます:
   <https://github.com/takanesano/school-scheduler/archive/refs/heads/main.zip>)
3. ダウンロードした ZIP ファイルを右クリック →
   「**すべて展開**」を選び、展開先はわかりやすい場所
   (例: `ドキュメント`)にします。
4. 展開してできたフォルダー(中に `setup.bat` や `app` という
   フォルダーが入っています)を開いてください。以後の作業は
   すべてこのフォルダーの中で行います。

> ZIP のままダブルクリックで開いて作業すると失敗します。
> 必ず「すべて展開」をしてから使ってください。

## 手順3: 初回セットアップ(最初の1回だけ)

1. フォルダーの中の **`setup.bat`** をダブルクリックします。
2. 「WindowsによってPCが保護されました」という青い画面が出たら、
   「**詳細情報**」→「**実行**」の順に押してください。
   (自分でダウンロードしたファイルなので実行して大丈夫です)
3. 黒い窓が開いて、文字がたくさん流れます。これは必要な部品を
   インターネットから取り込んでいるところです。
   **5〜10分くらいかかります。窓を閉じずに待ってください。**
4. 「**Setup complete!**」と表示されたら成功です。
   何かキーを押すと窓が閉じます。
   - かわりに「ERROR」と出たときは、下の
     「うまくいかないとき」を見てください。

> 豆知識: この手順では、フォルダーの中に `.venv` という専用の
> 「道具箱」(仮想環境と呼びます)を作って、必要な部品をそこに
> 入れています。パソコンの他の部分には影響しません。

## 手順4: 起動する(使うときは毎回これ)

1. **`start.bat`** をダブルクリックします。
2. 黒い窓が開き、数秒後にブラウザが自動で開いて、
   時間割の画面が表示されます。
   - もし「このサイトにアクセスできません」と出たら、数秒待って
     ブラウザの**再読み込み**(F5 キー)を押してください。
3. **黒い窓は閉じないでください。** この窓がソフトの本体です。
   閉じるとブラウザの画面も動かなくなります(小さくするのは OK)。

### 終了するとき

黒い窓を閉じるだけです(またはブラウザのタブを閉じてから黒い窓を
閉じる)。入力したデータは自動で保存されています。

### 2回目からは

`start.bat` をダブルクリックするだけです。手順1〜3は不要です。

## お試しデータを入れてみる(任意)

いきなり自分で入力する前に、動きを見てみたい場合は
**`sample-data.bat`** をダブルクリックしてください。
架空の生徒6人・先生4人・2週間の夏期講習のデータが入ります。
そのあと `start.bat` で起動し、「Schedule」タブの
「Generate schedule」ボタンを押すと時間割が自動生成されます。

> 注意: すでに自分のデータを入れている場合、お試しデータで
> 上書きされてしまうので実行しないでください。

## うまくいかないとき(Q&A)

**Q. setup.bat で「Python が見つからない」ような英語が出る**
手順1の「Add python.exe to PATH」のチェックを忘れた可能性が
大きいです。Python をアンインストール(設定 → アプリ →
Python → アンインストール)して、手順1をやり直してください。

**Q. setup.bat がすぐ閉じてしまう/ERROR と出る**
インターネットに接続されているか確認して、もう一度
`setup.bat` をダブルクリックしてください(途中からやり直せます)。

**Q. ブラウザに「このサイトにアクセスできません」と出る**
起動直後はソフトの準備が終わっていないことがあります。
黒い窓が開いたまま数秒待って、再読み込み(F5)してください。
黒い窓自体が閉じてしまっている場合は `start.bat` からやり直し。

**Q. 「ポートが使用中(address already in use)」のような英語が出る**
前回の黒い窓が閉じずに残っています。タスクバーを確認して古い
黒い窓を閉じてから、もう一度 `start.bat` を実行してください。

**Q. 入力したデータはどこに保存されている?**
フォルダーの中の `school.db` という1つのファイルです。
このファイルをコピーしておけばバックアップになります。
パソコンを買い替えるときも、このファイルをコピーすれば
データごと引っ越せます。

**Q. アンインストールしたいときは?**
フォルダーごと削除するだけです。レジストリなどは触りません。
(Python 自体を消したい場合は 設定 → アプリ から)

## 用語ミニ解説

- **コマンド**: 黒い窓(コマンドプロンプト)に打ち込む命令のこと。
  このガイドでは `.bat` ファイルのダブルクリックで代用しているので、
  自分で打ち込む必要はありません。
- **仮想環境(venv)**: このソフト専用の部品置き場。`setup.bat` が
  フォルダー内の `.venv` に自動で作ります。パソコン全体を汚しません。
- **サーバー**: `start.bat` で起動する黒い窓の中身。ブラウザは
  この「自分のパソコンの中のサーバー」に接続して画面を表示しています。
